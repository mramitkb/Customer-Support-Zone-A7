# Questions:

1) What is JSX and why is it used?
2) What is the difference between State and Props?
3) What is the useState hook, and how does it work?
4) How can you share state between components in React?
5) How is event handling done in React?

# Answers:

### 1st Answer:
`JSX` : JSX means JavaScript XML file, JSX copy the real DOM and just modify/update specific element where the actual changes happened.

<hr>

### 2nd Answer:
`State` : State contains data that change over time. <br> 
`Props` : Props are the things that we pass data from Parent to child component.

<hr>

### 3rd Answer:
`useState` : useState always contains a stateValue and a function-which changes the value when needed and a initial value assigned. Its a React Hook which allows component to store and update data. When state changes, its triggering re-render.

<hr>

### 4th Answer:
`Share State` : We can share `State` via `Props` drilling into the Parent Component to Child Component.

<hr>

### 5th Answer:
`Event Handling`: We can handle `event` with Clicking a Button, Submit a Form, typing in an Input field etc.
